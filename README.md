
📊 Customer Churn Prediction App (Deep Learning)
Developed by Usman Waris | Data Science,Machine Learning & Deep Learning Expert

🚀 Overview
This project is an end-to-end Deep Learning solution designed to predict and analyze customer attrition. Using an Artificial Neural Network (ANN), the application processes customer data to identify those at risk of leaving, enabling businesses to take proactive retention measures.

🔗 Live Demo
Access the interactive dashboard here:[https://churnpredictiondlmodel-c9bhkemuuh3adepcs3ysza.streamlit.app/] 

✨ Key Features
Deep Learning Engine: Built using a multi-layer Artificial Neural Network (ANN) for high-precision classification.

Intelligent Risk Segmentation: Automatically categorizes customers into three priority levels:

🔴 High Risk: Immediate attention required.

🟠 Moderate Risk: Needs engagement.

🟢 Low Risk: Loyal/Stable customers.

Explainable AI (XAI): Integrated with LIME (Local Interpretable Model-agnostic Explanations) to understand the "Why" behind every prediction.

Actionable Reporting: One-click feature to download the High-Risk Customer List as a CSV for marketing and sales teams.

Interactive UI: A clean and responsive dashboard built with Streamlit.

🛠️ Tech Stack
Language: Python 3.10+

Deep Learning: TensorFlow / Keras

Data Processing: Pandas, NumPy, Scikit-learn

Visualization: Plotly, Seaborn, Matplotlib

Deployment: Streamlit Cloud
